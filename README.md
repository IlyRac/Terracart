## âš™ï¸ How to use
1. Craft the Terracart and **choose a color** (optional) before placing it.
2. **Smelt** a colored cart in a furnace to return it to the default cart item.
   - **Note:** carts **do not** drop themselves. To change a cartâ€™s color, choose it before placing; otherwise you must craft a new cart.
3. Right-click to mount.
4. Use normal movement keys to drive.
5. **Refuel** with **coal** while moving (fuel drains only while moving).
